Base element

specifies the base URL and/or target for all relative URLs on the page. Must have a href/target attribute (or both) and there must only be one base element defined.

Responsiveness

to make a page responsive, add the below meta tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
